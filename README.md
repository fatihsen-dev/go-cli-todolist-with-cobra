## Go Cli Todolist - Cobra cli

### install & run
```shell
git clone https://github.com/fatihsen-dev/go-cli-todolist-with-cobra.git
cd go-cli-todolist-with-cobra
go run main.go OR go run .
```
<br/>

### Cli commands
<hr/>

### Create
```
⮞  go run . create Todo Item 1
+-------+--------------+---------+---------------------+
|  ID   |     TODO     | STATUS  |        DATE         |
+-------+--------------+---------+---------------------+
| 12147 | Todo Item 1  | pending | 21-09-2023 11:12:32 |
+-------+--------------+---------+---------------------+
```

### Update
```
⮞  go run . update -i 36786 -n complated
+-------+--------------+-----------+---------------------+
|  ID   |     TODO     |  STATUS   |        DATE         |
+-------+--------------+-----------+---------------------+
| 36786 | Todo Item 1  | complated | 21-09-2023 11:13:10 |
+-------+--------------+-----------+---------------------+
```

### Delete
